## Comandos de setup en ubuntu

Crear entorno virtual:

```cmd
python3 -m venv venv
```

Activar entorno virtual en ubuntu

```cmd
source venv/bin/activate
```

Ver librerias instaladas:

```cmd
pip freeze
```

Instalar requirements.txt

```cmd
pip install -r requirements.txt
```
